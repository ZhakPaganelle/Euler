Для целого M определим R(M) как сумму 1/(p·q) для всех пар целых чисел p и q, которые удовлетворяют следующим условиям:


 1 ≤ p < q ≤ M
 p + q ≥ M
 p и q - взаимно простые числа.


Также определим S(N) как сумму R(i) для 2 ≤ i ≤ N.
